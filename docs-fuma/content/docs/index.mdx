A minimal, provider-agnostic agentic loop. Every fundamental piece sits behind
an interface, so it can be swapped without touching the loop — the point of the
package is to make the **core things plug-and-play and independently testable**.

## Why it exists

- **Plug-and-play seams** — the model client, memory, tools, and stop conditions
  are all interfaces. Implement them with a plain object or function.
- **Streaming by default** — `ModelClient.stream()` returns an async iterable of
  `StreamEvent`s, so reasoning, text, and tool calls arrive incrementally.
- **Provider-agnostic** — nothing in the core references a specific LLM SDK. Any
  OpenAI-compatible endpoint works with a raw `fetch`.
- **Independently testable** — each seam is verified in isolation with
  deterministic test doubles; zero network.
